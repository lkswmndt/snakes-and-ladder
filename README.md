# Adu Cerdas di Jalur Ular: Langkah Pasti Menuju Juara

## Deskripsi Game
Adu Cerdas di Jalur Ular: Langkah Pasti Menuju Juara adalah game edukasi berbasis Python dan Pygame yang menggabungkan permainan ular tangga dengan kuis pengetahuan umum.

Pemain harus melempar dadu untuk bergerak menuju kotak 100 sambil menjawab pertanyaan pilihan ganda. Jawaban yang benar memberikan kesempatan tambahan, sedangkan jawaban salah akan memberikan hukuman mundur beberapa langkah.

Game ini dibuat untuk melatih:
- Pengetahuan umum
- Logika berpikir
- Ketelitian
- Pembelajaran sambil bermain

---

## Fitur Game
- 2 Player Mode
- Sistem ular dan tangga
- Pertanyaan pilihan ganda (ABCD)
- Main menu interaktif
- Halaman How To Play
- Halaman About Us
- Sistem menang dan restart
- Tampilan visual menggunakan Pygame
